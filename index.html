Here we will discuss about the basic git codes.

  1. for initialising : git init
  2. for adding a local file to git : git add .
  3. for commiting a file to git : git commit -m "first commit"
  4. for saving changes to origin : git remote add origin https://github.com/username/repo.git
  5. for pushing the update : git push -u origin master
  6. for pulling the update : git pull origin master
  7. for removing a file from git : git rm -- file_name
  8. for creating a git branch : git branch branch-name
  9. for switching to a branch : git checkout branch-name
  10. for merging a branch : git merge branch-name
  11. for deleting a branch : git branch -d branch-name
  12. for undoing last commit : git reset --hard HEAD~1
