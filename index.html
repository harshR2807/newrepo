Here we will discuss about different steps taken for creating a Repo in Git.
first we create a local repository. Then we add a file.
After adding a file we commit the changes. Next we save the changes on our Git Repo and push the file.
Here's the code for the abhove changes:

  1. for initialising : git init
  2. for adding a local file to git : git add .
  3. for commiting a file to git : git commit -m "first commit"
  4. for saving changes to origin : git remote add origin https://github.com/username/repo.git
  5. for pushing the update : git push -u origin master
